Introduction

The Scratch Node Client is a module for Node.JS that allows communication with the Scratch 3.0 server. With it, you can view and post comments, view and set cloud variables, and more.

What you should already know

To use Node Scratch Client you need at least a little knowledge of JavaScript. You can learn this programming language on free courses like Codecamp.

Setting it up

You can create a Node.JS program online on replit.com, by clicking the plus sign and selecting Node.JS as the programming language. To use Scratch Node Client in your program, enter the following line of code at the top: 

const scratch = require("scratch-node-client");

The module should be installed automatically the first time you run the program.

Logging in

To log in to your Scratch account, do this, replacing YOUR_USERNAME and YOUR_PASSWORD for your username and password respectively: 

const Client = new scratch.Client({
	username: "YOUR_USERNAME",
	password: "YOUR_PASSWORD"
});

Client.login().then(() => {
	// the code that will execute after logging will go here
});

WARNING: if you are using an online service like replit you must not put your password there directly since the code there is public and anyone could see it. Instead, you should save your password as a secret which you can access through the padlock icon. If you create a secret with the key PASSWORD and the value as your password, it will be safe from everyone. You can then access its value using process.env.PASSWORD.
